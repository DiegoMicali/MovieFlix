MovieFlix API
Overview
A robust RESTful API built to manage a catalog of movies, streaming platforms, and categories. Designed to demonstrate proficiency in modern Java backend development, secure architecture, and relational database management.

Technical Stack
Ecosystem: Java, Spring Boot, Spring Security

Authentication: JWT (JSON Web Token)

Database: PostgreSQL, SQL

Data Migration: Flyway

Infrastructure: Docker

Documentation: Swagger / OpenAPI

Technical Highlights
Secure Authentication: Implemented stateless user authentication and authorization utilizing Spring Security and JWT.

Database Modeling: Engineered complex table relationships (Movies, Categories, Streaming) in PostgreSQL, maintaining strict database versioning through Flyway.

Resilient API Design: Developed structured RESTful controllers with centralized exception handling to guarantee predictable and standardized HTTP responses.

Interactive Documentation: Exposed self-updating API documentation via Swagger for seamless frontend integration testing.
